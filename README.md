# نتائج الطلاب 2026

مرحباً بك في نظام الحصول على نتائج الطلاب

## 📋 عن النظام
نظام ويب متقدم للحصول على نتائج الطلاب بكل سهولة وأمان. يوفر واجهة سهلة الاستخدام للطلاب والمسؤولين.

## 🎯 المميزات
- 🔐 نظام آمن للدخول
- 📊 عرض النتائج بشكل واضح
- 👨‍💼 لوحة تحكم للمسؤولين
- 📱 واجهة سهلة وسريعة

## 🚀 كيفية الاستخدام

### للطلاب:
1. افتح الموقع من الرابط أدناه
2. أدخل رقم الطالب وكلمة السر
3. شاهد نتائجك مباشرة

### للمسؤولين:
1. ادخل إلى لوحة التحكم
2. أدخل بيانات الدخول الإدارية
3. أضف أو عدّل النتائج

## 📁 محتويات المشروع
- `index.html` - صفحة دخول الطلاب
- `admin.html` - صفحة لوحة التحكم
- `app.js` - وظائف البرنامج الرئيسية
- `admin.js` - وظائف إدارة النتائج
- `style.css` - تنسيق الموقع
